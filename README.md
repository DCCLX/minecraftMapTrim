# minecraftMapTrim
Простой скрипт для генерации списка .mca файлов на которых есть заприваченные постройки.

mysql хранилище не поддерживается

`-w` - путь до файла со списком регинов WorldGuard

Пример вывода:
```
r.3.1.mca
r.14.-14.mca
r.4.1.mca
r.9.7.mca
r.13.-32.mca
r.4.18.mca
r.10.7.mca
r.-21.-66.mca
r.-5.-19.mca
r.14.11.mca
r.10.-41.mca
```
